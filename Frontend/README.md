# 缠论分析前端

基于React、Lightweight Charts 和 Element UI 的缠论分析前端界面。

## 安装依赖

```bash
npm install
```

## 开发运行

```bash
npm run dev
```

应用将在 http://localhost:3000 启动

## 构建生产版本

```bash
npm run build
```

构建后的文件将在 `dist` 目录中

## 功能特性

- K线图展示（基于 Lightweight Charts）
- 笔、线段、中枢可视化
- 买卖点标注
- 响应式设计
- 简洁的用户界面

## 使用说明

1. 在顶部输入框输入股票代码（例如：sz.000001）
2. 选择开始时间和结束时间（结束时间可选）
3. 点击"查询"按钮
4. 系统将展示K线图和缠论分析结果
